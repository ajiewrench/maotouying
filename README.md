## 简介

“猫头鹰 · 局域网文件互传助手” 是一个轻量级、快速、直观的局域网文件传输工具，适合个人和小团队在同一 Wi-Fi 或局域网内快速分享文件。  
软件界面简洁，支持拖选文件、二维码扫描、URL 点击下载，追求专业、可信和高效。

特点：

- Windows 下可直接运行 EXE  
- 上传文件后自动生成可扫码下载二维码  
- 文件 URL 可直接点击打开浏览器  
- 上传有效期 10 分钟  
- 界面小巧、快速、易用

---

## 功能

1. **文件选择**：支持选择本地文件（图片、PDF、压缩包等）  
2. **上传**：生成局域网可访问 URL 并显示二维码  
3. **二维码下载**：手机扫码即可下载文件  
4. **倒计时显示**：上传链接有效期剩余时间实时显示  
5. **重置**：可以清空当前文件，重新上传  

---

## 运行

电脑端点击EXE文件。选择文件，上传，。手机扫码就可以下载上传好的文件
